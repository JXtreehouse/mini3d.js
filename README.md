# mini3d.js
一个实验性的WebGL项目，旨在使用WebGL从头实现一些渲染效果，并尽量提供可用性，以在一些网页展示/游戏中使用。

* 在线示例: http://happyfirecn.gitee.io/mini3djs_examples/
* 开发笔记：https://blog.csdn.net/n5/category_9647112.html

## 预计支持的功能
* 场景和变换
* 多光源前向渲染
* 延迟渲染
* 系统Shader框架
* 自定义Shader/材质
* 模型载入
* ShadowMap阴影
* 后期处理框架
* 水体渲染
* 骨骼动画
* 实例化绘制
* 动态Batch
* 其他
